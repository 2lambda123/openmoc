Detailed Description

An MOCKernel object specifies a functionality to apply to MOC segments.

An MOCKernel is an object that owns some specified data and contains an "execute" function which applies some functionality to the data. This is useful in MOC where it is very common to apply some function to segment data in either a nested loop structure or from on-the-fly calculations. Kernels specify the actions applied to the segments, reducing the need for repeated code. This class is the parent class of CounterKernel and VolumeKernel. A generic MOCKernel should not be explicity instantiated. Instead, an inheriting class should be instantiated which implements the "execute" function.
