Vector::Vector ( int  num_x = 1,
int  num_y = 1,
int  num_groups = 1 
)

Constructor initializes Vector object as a floating point array and sets the vector dimensions. The vector is ordered by cell (as opposed to by group) on the outside to be consistent with the Matrix object. Locks are used to make the vector object thread-safe against concurrent writes the same value. One lock locks out multiple rows of the vector at a time representing multiple groups in the same cell.

Parameters
num_xThe number of cells in the x direction.
num_yThe number of cells in the y direction.
num_groupsThe number of energy groups in each cell.
