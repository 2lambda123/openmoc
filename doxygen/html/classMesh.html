Detailed Description

Represents a cartesian mesh for Coarse Mesh Finite Difference Diffusion acceleration.

Constructor & Destructor Documentation

bool  cmfd_on = false,
double  relax_factor = 0.6,
int  mesh_level = -1 
)

Constructor initializes boundaries and variables that describe the Mesh.

The construcor initializes the many variables to zero, initializes cmfd acceleration to off, and initializes the boundary conditions to REFLECTIVE.

Parameters

Initializes the Mesh by allocating memory for various arrays.

This method is called by the geometry once the width of the Mesh has been determined. This method allocates memory for the volumes, old and new flux arrays, lengths, bounds, and cell FSR vectors.

void Mesh::splitCorners ( )

Split the currents of the Mesh cell corners to the nearby surfaces.

left bottom corner -> bottom surface and left surface of mesh cell below; right bottom corner -> bottom surface and right surface of mesh cell below; right top corner -> right surface and top surface of mesh cell to the right; left top corner -> left surface and top surface of mesh cell to the left.
