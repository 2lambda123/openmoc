int CellBasic::getNumFSRs ( )
virtual

Return the number of flat source regions in this CellBasic.

This method is used when the Geometry recursively constructs flat source regions. By definition, CellBasics are the lowest level entity in the geometry and thus only have one flat source region within them, so this method always returns 1.
