Detailed Description

The plotter module provides utility functions to plot data from OpenMOCs C++ classes, in particular, the geomery, including Material, Cells and flat source regions, and fluxes and pin powers.

def openmoc.plotter.plot_cells (   geometry,
  gridsize = 250 
)

This method takes in a Geometry object and plots a color-coded 2D surface plot representing the Cells in the Geometry.

The geometry object must be initialized with Materials, Cells, Universes and Lattices before being passed into this method. A user may invoke this function from an OpenMOC Python file as follows:

openmoc.plotter.plot_cells(geometry)
Parameters
geometrya Geometry object which has been initialized with Materials, Cells, Universes and Lattices
gridsizean optional number of grid cells for the plot

def openmoc.plotter.plot_segments (   track_generator)

Plots the characteristic Track segments from an OpenMOC simulation.

This method requires that tracks have been generated by a TrackGenerator object. Each segment is colored by the ID of the unique flat flat source region it is within. A user may invoke this function from an OpenMOC Python file as follows:

openmoc.plotter.plot_segments(track_generator)
Parameters
track_generatorthe TrackGenerator which has generated Tracks
