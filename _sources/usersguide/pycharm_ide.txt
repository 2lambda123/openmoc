======================================
Integrated Development Environment IDE
======================================

-----------
PyCharm IDE
-----------

The `PyCharm`_ IDE is recommended for OpenMOC users developing input and data processing scripts in Python. Although PyCharm is a commercial product, a community version is provided for free with many of the most essential features including the following:

* Syntax highlighting
* Auto-indentation and code formatting
* Code completion
* Line and block commenting
* Refactoring
* Integrated debugger
* Embedded Python interpreter
* Embedded version control
* Issue tracker integration



Installation
------------

PyCharm can be `downloaded and installed`_ on Windows, Mac OS X, and Linux.
