CellBasic¶

class CellBasic

Represents a cell filled with a material.

Public Functions

CellBasic(short int universe, short int material, int rings = 0, int sectors = 0, short int id = 0)

Constructor sets the user-specified and unique IDs for this cellbasic.

Parameters
  • universe -

    the ID for the universe within which this cellbasic resides

  • material -

    the ID for the material which fills this cellbasic

  • rings -

    the number of equal volume rings to divide this cell into (the default is zero)

  • sectors -

    the number of angular sectors to divide this cell into (the default is zero)

  • id -

    the user-specified cell ID

int getNumFSRs()

Return the number of flat source regions in this cellbasic.

This method is used when the geometry recursively constructs flat source regions. By definition, cellbasic’s are the lowest level entity in the geometry and thus only have one flat source region within them, so this method always returns 1.

Return
the number of FSRs in this cell
