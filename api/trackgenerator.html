class TrackGenerator

The track generator is dedicated to generating tracks which cyclically wrap across the geometry.

The track generator creates track and initializes boundary conditions (vacuum or reflective) for each track.

void generateTracks()

Generates tracks for some number of azimuthal angles and track spacing *.
