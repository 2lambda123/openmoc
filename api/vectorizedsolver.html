class VectorizedSolver

This is a subclass of the CPUSolver class which uses memory-aligned data structures and Intel’s auto-vectorization.

Public Functions

VectorizedSolver(Geometry * geometry = NULL, TrackGenerator * track_generator = NULL)

Constructor initializes empty arrays for source, flux, etc.

The construcor retrieves the number of energy groups and flat source regions and azimuthal angles from the geometry and track generator, and uses this to initialie empty arrays for the flat source regions, boundary angular fluxes, scalar flatsourcergion fluxes, flatsourceregion sources and flatsourceregion powers. The constructor initalizes the number of threads to a default of 1.

Parameters
  • geometry -

    an optional pointer to the geometry

  • track_generator -

    an optional pointer to the trackgenerator

~VectorizedSolver()

Destructor deletes arrays of boundary angular flux for all tracks, scalar flux and source for each flat source region.
