Quick Install Guide¶

This quick install guide outlines the basic steps needed to install OpenMOC on your computer. For more detailed instructions on configuring and installing OpenMOC, see Installation and Configuration in the User’s Manual.







Installing from Source on Linux or Mac OS X¶


All OpenMOC source code is hosted on GitHub. To download and install OpenMOC, you need to install git and GNU’s C++ compiler, g++_. In addition, you need to install Python 2.6 or 2.7 and the following Python packages on your machine: swig, setuptools, h5py, numpy, and matplotlib. These packages can easily be installed using a package manager for Linux and Mac OS (see Installation and Configuration for more details).


You can download and install OpenMOC by entering the following commands in a terminal:


git clone git://github.com/mit-crpg/OpenMOC.git
cd OpenMOC
sudo python setup.py install





This will build a shared library accessible as a Python package named openmoc and install it (by default in /usr/local/lib/python2.7/dist-packages). The openmoc Python package can now be imported into any Python script as follows:


import openmoc
