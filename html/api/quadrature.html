class Quadrature

Stores values for a variety of polar quadratures which may be used.

Public Functions

Quadrature(quadratureType type = TABUCHI, int num_polar_angles = 3)

Constructor initializes arrays with polar angles and weights.

Parameters
  • type -

    the quadrature type (LEONARD or TABUCHI)

  • num_polar_angles -

    the number of polar angles (1, 2, or 3)

~Quadrature()

Destructor deletes arrray of sines of the polar angles, the weights of the polar angles and the products of the sines and weights.

int getNumPolarAngles()

Returns the number of polar angles (1, 2 or 3).

the number of polar angles

Return

std::string toString()

Converts this quadrature to a character array of its attributes.

The character array includes the polar quadrature type, the number of polar angles, and the values of the sine of each polar angle, the weight of each polar angle, and the product of the sine and weight of each polar angle.

Return
a character array of this quadrature’s attributes
